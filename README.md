# douyin-live-hourly-order-report
# 小时订单报1.0  # 小时订单报1.0

一个用于抖音电商/巨量百应直播间的 Windows 桌面应用。

应用会在设定时间自动打开巨量百应，进入直播明细中的直播大屏，切换到商品页并读取直播间订单，按“运行时间前 1 小时”汇总订单数据，生成商品件数、金额和总 GMV，并通过钉钉机器人发送到群里。

## 功能

- 支持手动运行和定时运行
- 支持开机自动启动
- 未登录百应时自动等待登录
- 按运行时间前 1 小时抓取订单
- 支持按订单时间自动翻页读取
- 支持商品名称归类
- 支持钉钉机器人连通性检查
- 抓取完成后自动关闭打开的浏览器页面

## 使用场景

适合直播运营在固定时间自动统计上一小时直播间成交商品、成交件数、成交金额和总 GMV。
